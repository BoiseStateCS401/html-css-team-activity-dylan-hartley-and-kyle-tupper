HTML

HTML creates the basic framework of a website. Meaning that it will create the layout and the design of a website including changing font, creating titles, and the background color of the website. This will create the actual content of the website and define how it will look in the browser.

CSS

CSS is concerned with the style and components that deal with the presentation of information. For example, layout, colors, fonts, for all headings or text on the page. All styles used to be done directly inside of HTML but CSS allows for greater control over the look and feel of a website for developers. CSS is the most compatible styling resource available.



CSS: Cascading Style Sheets

JavaScript

JavaScript is a web-based, interpreted, object-oriented programming language used to create interactable content or do things that HTML alone is unable to do. Using JavaScript, you can edit content on the page, listen to user events and respond accordingly, or fetch and display data from a web server.
